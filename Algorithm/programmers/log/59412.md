#### [입양 시각 구하기(1)](https://programmers.co.kr/learn/courses/30/lessons/59412)


###### My Solution #1

MySQL
```sql
SELECT HOUR(DATETIME) AS 'HOUR', COUNT(ANIMAL_ID) AS 'COUNT'
    FROM ANIMAL_OUTS
    WHERE HOUR(DATETIME) BETWEEN 9 AND 19
    GROUP BY HOUR(DATETIME)
    ORDER BY HOUR(DATETIME);
```
