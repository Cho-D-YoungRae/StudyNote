#### [18406. 럭키 스트레이트](https://www.acmicpc.net/problem/18406)

**My solution**
```python
if __name__ == '__main__':
    # 입력받은 점수 숫자 하나씩 정수로 변환하여 리스트로
    N = list(map(int, list(input())))

    # 리스트 반씩 더한다.
    my_status = "LUCKY" if sum(N[:len(N)//2]) == sum(N[len(N)//2:]) \
                                                        else "READY"

    print(my_status)
```
