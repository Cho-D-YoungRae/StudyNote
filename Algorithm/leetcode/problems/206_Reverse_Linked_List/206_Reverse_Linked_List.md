#### [206_Reverse_Linked_List](https://leetcode.com/problems/reverse-linked-list/)
> 219pg


###### My Solution 1
```python
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        result = None
        
        while head:
            head.next, head, result = result, head.next, head
            
        return result
```
