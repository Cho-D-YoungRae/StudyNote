# hist()
> 데이터 형태를 빠르게 검토해보기 위해 각 숫자형 특성을 히스토그램으로 그려볼 수 있다.

```python
import matplotlib.pyplot as plt

housing.hist(bins=50, figsize=(20, 15))
plt.show()
```


    
![](./images/데이터%20구조%20훑어보기/output_8_0.png)

## 알 수 있는 내용
1. `median_income` 특성이 US달러로 표시되어 있지 않은 것 같다. 상한이 15(15.0001), 하한이 0.5(0.4999)가 되도록 되어있다고 한다. 머신러닝에서는 전처리된 데이터를 다루는 경우가 흔하다. 이것이 어떻게 계산된 것인지 이해하고 있어야 한다.
2. `housing_median_age`, `median_house_value` 최댓값과 최솟값이 한정되어있다. `median_house_value`의 경우에는 타깃 속성으로 사용되기 때문에 문제가 될 수 있다. 가격이 한곗값을 넘어가지 않도록 머신러닝 알고리즘이 학습될 수 있다. 500000(상한)을 넘어가더라도 정확한 예측값이 필요하다면 우리가 선택할 수 있는 방법은 2가지다.
	- 한곗값 밖의 구역에 대한 정확한 레이블을 구한다.
    - 훈련세트에서 이런 구역을 제거한다.
3. 특성들의 스케일이 다르다.
4. 많은 히스토그램의 꼬리가 두껍다. 가운데에서 왼쪽보다 오른쪽으로 더 멀리 뻗어 있다. 이런 형태는 일부 머신러닝 알고리즘에서 패턴을 찾기 어렵게 만들기 때문에 종 모양 분포가 되도록 변형시키는 것이 좋다.
